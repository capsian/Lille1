1.1 Puissance n

On veut montrer que E(Xn) ≥ E(X)n lorsque n est paire.

Pour montrer cette inégalité, nous allons utiliser l'inégalité de Jensen. Il s'agit alors de montrer que fn : x ↦ xn est convexe.

Comme n est pair, il existe k tel que n = 2k. \(f_n''(x) = x^{n-2} = x^2{k-1}\). Ainsi pour tout x la dérivée seconde de fn est positive, ce qui implique que fn est convexe.

1.2 Jensen discret

1.2.1 Récurrence pour montrer l'identité

On montre par récurrence sur k que : \[f\left( \sum_{i=1}^k \lambda_i x_i \right) \leq \sum_{i=1}^k \lambda_i f(x_i)\]

Lorsque k=1, par hypothèse λ₁ = 1 et l'identité devient f(x₁) ≤ f(x₁) ce qui est trivial.

Supposons que cette identité est vraie pour k et montrons la pour k+1. \[f\left( \sum_{i=1}^{k+1} \lambda_i x_i \right) = f\left(\left( \sum_{i=1}^{k} \lambda_i x_i \right) + \lambda_{k+1}x_{k+1} \right)\] On pose \[a = \frac{\sum_{i=1}^k \lambda_i x_i}{1 -\lambda_{k+1}}\] on obtient : \[f\left( \sum_{i=1}^{k+1} \lambda_i x_i \right) = f\left((1-\lambda_{k+1}) a + \lambda_{k+1}x_{k+1} \right)\] par convexité de f, nous avons : \[f\left( \sum_{i=1}^{k+1} \lambda_i x_i \right) \leq (1-\lambda_{k+1}) f(a) + \lambda_{k+1}f(x_{k+1})\]

Pour conclure, il nous faut montrer que \[f(a) =f\left( \sum_{i=1}^k\frac{\lambda_i}{1-\lambda_{k+1}} x_i\right) \leq \sum_{i=1}^k\frac{\lambda_i}{1-\lambda_{k+1}} f(x_i)\] Pour qui cette inégalité soit vraie par récurrence, il suffit de montrer que : \[ \sum_{i=1}^k\frac{\lambda_i}{1-\lambda_{k+1}} = 1\] Or par hypothèse \(\sum_{i=1}^{k+1} \lambda_i = 1\). Ainsi, \(\sum_{i=1}^{k} \lambda_i = 1-\lambda_{k+1}\) d'où l'égalité recherchée.

En utilisant l'inégalité \[f(a) \leq \sum_{i=1}^k\frac{\lambda_i}{1-\lambda_{k+1}} f(x_i)\] on obtient : \[\begin{array}{rcl} f\left( \sum_{i=1}^{k+1} \lambda_i x_i \right) &\leq& (1-\lambda_{k+1})f(a) + \lambda_{k+1}f(x_{k+1})\\ &\leq& (1-\lambda_{k+1}) \sum_{i=1}^k\frac{\lambda_i}{1-\lambda_{k+1}} f(x_i) + \lambda_{k+1}f(x_{k+1})\\ &=&\sum_{i=1}^{k+1}\lambda_i f(x_i) \end{array}\]

1.2.2 Inégalité liée à l'espérance

On sait que X prend un nombre fini de valeur. Disons que ces valeurs sont dans l'ensemble {v1, … vk}. Par définition, \[E(f(X)) = \sum_{i=1}^k Pr(X=v_i)f(v_i)\] Comme \(\sum_{i=1}^k Pr(X=v_i) = 1\), d'après la question précédente, nous avons :

\[E(f(X)) = \sum_{i=1}^k Pr(X=v_i)f(v_i) \geq f(\sum_{i=1}^k Pr(X=v_i) v_i) = f(E(X))\]

1.3 Jets de dés

1.3.1 Question 1

On veut calculer : E(X | X₁ = 2) Or par définition \[E(X \mid X_1 = 2) = \sum_{k=2}^{12} k Pr(X = k \mid X_1 = 2) = \sum_{k=3}^8 k Pr(X = k \mid X_1 = 2) = \frac{1}{6} \sum_{k=3}^8 k = \frac{33}{6} =\frac{11}{2}\]

1.3.2 Question 2

On veut calculer : E(X₁ | X = 5) Par définition : \[\begin{array}{rcl} E(X_1 \mid X = 5) &=& \sum_{k=1}^6 kPr(X_1 = k \mid X = 5) \\ &=& \sum_{k=1}^6 k\frac{Pr(X_1 = k \cap X = 5)}{Pr(X_1 = k)}\\ & =& \sum_{k=1}^6 6k Pr(X_1 = k \cap X = 5)\\ &= & \sum_{k=1}^6 6k Pr(X_1 = k \cap X_2 = 5-k)\\ &=& \sum_{k=1}^4 6k Pr(X_1 = k)Pr(X_2 = 5-k)\\ & = &\frac{1}{6} \sum_{k=1}^4 k\\ & = &\frac{10}{6}\\ &= &\frac{5}{3} \end{array}\]
