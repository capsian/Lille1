IV/ COMMUTATEUR

	q1/
			Le switch retransmet les données à leur destinataires. (switch = hub intélligent :p )
			Meilleur bande passante
	q2/
			Le switch reçoit les packet réseau et associe les adresses sources avec le port par lequel ils ont reçu le packet.
			Pour choisir le port de destination d'un packet, le switch recherche l'adresse de destination dans la table et envoi le packet uniquement au port concerné. 
	q3/
			Oui il a besoin de connaître les adresses MAC afin de pouvoir identifier précisément les machines concernés, ils n'a pas besoin de connaître les adresses IP des paquets car il s'agit d'un réseau
			local.
			Les switchs interviennent sur la couche liaison.
			les hubs interviennent sur la couche physique.
	q4/
			Le switch permet d'envoyer les paquets uniquement à la machine concerné.
			Pas de collisions/erreurs
	q5/
			Broadcast : Tous le mondre reçoit le ping. Tous le mondre répond.
